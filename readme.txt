Steam创意工坊订阅地址：http://steamcommunity.com/sharedfiles/filedetails/?id=807268921

饥荒对战 猪王争霸：
你们要的饥荒分组对战来了，玩腻了饥荒无止境的生存，不妨来试试这个MOD吧. 
喜欢这个MOD的朋友不要忘了点个赞哦，你们的支持就是我们继续搞事情的动力...
建议在无尽-PVP模式下开启该MOD，游戏中按Ctrl+F键攻击敌方成员.

这次更新内容：加快节奏
1.分组的路牌传送，每个队伍都有自己的传送路牌，每队允许拥有的路牌总数为5
2.金子可以雇佣猪人
3.队伍标识，按Tab键查看玩家所属的队伍名称
4.增加一些动物的击杀得分
5.整合属性显示和快速采集功能，不需要加其他辅助MOD
5.修复一些BUG

上次更新功能简介：
1.添加猪王和猪人升级，得分可升级猪王，升级后该阵营的猪人也会升级，领土面积也会随之扩大 
2.添加猪王防御塔(数量可设置) 
3.添加B键回城功能 
4.猪王被攻击提示 
5.权限和平衡完善 
6.和平期，具体时间可在MOD设置页面设置
7.猪王是否可杀，可在MOD设置中设置
8.更多惊喜请自行探索 

BUG反馈QQ群486149251

胜利条件
获取到足够高的分数 
杀掉其他阵营的猪王，只剩自己的猪王
规则很简单，两个条件任意达成其一就能取得最后的胜利！ 

提示
将一些有价值的物品给自己的猪王可以换取分数 
击杀其他阵营玩家和部分怪物可以换取分数 
猪王附近的建筑和农作物会被保护 
自己阵营的猪王被杀时阵营会被解散，财产将被击杀势力占有(也可设置猪王不可杀) 
想要赢得比赛，前提是要保护好自己的猪王，发展和武力并存 
共4个阵营供玩家选择，MOD设置页面可设置阵营数量，PVP模式下同阵营玩家不能相互攻击 
可指定选择队伍，也可随机队伍，可在MOD设置里进行设置(长期服务器建议随机队伍，开黑玩则指定队伍) 
前期将会有一段和平发展期，不能攻击敌方成员(具体时间可在MOD设置中修改)

建议在PVP模式下开启该MOD，也可把猪王设置成不可杀，单纯地竞技得分
